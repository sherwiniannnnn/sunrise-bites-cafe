Database Connection Test

Test the connection between your frontend and Neon PostgreSQL database.

Setup Instructions

  1. Create a Neon PostgreSQL database at neon.tech
  2. Run the SQL setup scripts in your database
  3. Set environment variables in Netlify:
    • DATABASE_URL - Your Neon database connection string
  4. Deploy to Netlify with serverless functions
  5. Test the connection using the buttons above
